A [[machine learning]] apparoach consolidated by [[Yann LeCun]], [[Yoshua Bengio]], and [[Geoffrey Hinton]] (2015).

Multi-layered (i.e. "***deep***") [[Artificial Neural Network]] can in theory approximate any function given the  correct configurations, thanks to the [[Universal Approximation Theorem]]. ***Deep Learning*** describes the process of arriving at a good approximation given an artificial neural network and training data for the function the network seeks to approximate via the [[backpropagation]] algorithm.
