**Idea**
Multi-layered (i.e. "***deep***") [[Artificial Neural Network]] can in theory approximate any function given the  correct configurations, thanks to the [[Universal Approximation Theorem]]. The ***Deep Learning*** describes the process of arriving at this approximation given an artificial neural network and training data for the function the network seeks to approximate via the [[backpropagation]] algorithm.
