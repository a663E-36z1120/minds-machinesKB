**Machine learning (ML)** is a subfield of [[artificial intelligence]] where systems learn patterns from data instead of being explicitly programmed. Rather than writing rules by hand, developers provide examples, and the algorithm finds statistical relationships to make predictions or decisions.

**Key features of ML algorithms:**
- **Learns from data**: Improves performance over time as it sees more examples.
- **Generalizes**: Applies learned patterns to new, unseen data.

**Main Types**:
- **Supervised learning**: Learns from labeled data.
- **Unsupervised learning**: Finds patterns in unlabeled data (e.g., clustering).
- **[[Reinforcement learning]]**: Learns through trial and error via rewards.
